# 実装済み機能まとめ (2026-03-14 時点)

### 🛡️ 認証・セキュリティ設定
*   **JWTベースの認証システム**: ログイン・ログアウト・セッション維持機能。
*   **ロールベースアクセス制御 (RBAC)**: `HQ_ADMIN`（本部）、`STORE_MANAGER`（店長）、`OFFICE_STAFF` などの権限管理。
*   **操作ログ保存**: システム内の主要な操作（ログイン、更新など）を監査ログとして記録。

### 📢 全社お知らせ機能 (Announcements)
*   **ホームフィード**: 投稿されたお知らせをタイムライン形式で表示。
*   **カテゴリ管理**: 「社長メッセージ」「重要」「必須」などのカテゴリ分け。
*   **画像・カバー表示**: 投稿へのカバー画像添付とリストでの視覚化。
*   **リアクション・交流**:
    *   **いいね機能**: 各投稿への「いいね」付与。
    *   **コメント機能**: 投稿に対する従業員同士のコメント。
    *   **既読管理**: 各ユーザーが記事を読んだかどうかのトラッキング。
*   **管理者向け統計**: 記事ごとの既読数や既読率の確認。

### 🧱 店舗ポータル・メニュー項目 (Widgets)
*   **タイル形式のメニュー**: サイドバーやポータルに表示されるインタラクティブなタイル。
*   **多様なウィジェットタイプ**:
    *   **アンケート (SURVEY)**: 質問と選択肢による簡易収集。
    *   **出欠確認 (ATTENDANCE)**: イベント等の参加可否回答。
    *   **チェックリスト (CHECKLIST)**: タスクの完了確認。
    *   **掲示板 (BOARD)**: ユーザーが自由に投稿できるボード。
    *   **外部リンク (LINK)**: 他サービスへのショートカット。
*   **高機能ビルダー**: ドラッグ＆ドロップによるタイルの配置・サイズ（S/M/L）・公開期限の設定。
*   **回答収集**: アンケートや出欠の集計結果を管理者画面で閲覧・管理。

### 📩 問い合わせ・コミュニケーション
*   **問い合わせ管理**: 従業員から本部への問い合わせ送信。
*   **ステータス管理**: 「対応中」「完了」などの状態変更。
*   **メッセージスレッド**: 問い合わせ内容に基づいた本部との個別メッセージのやり取り。

### 🏢 組織・マスター管理
*   **ユーザー管理**: 従業員情報の作成・編集・削除。
    *   **CSV一括登録**: 従業員データの大規模なインポート機能。
*   **店舗・グループ管理**: 組織構造（店舗、エリア、本部部署など）の階層管理。
    *   **店舗グループ**: 特定のエリア限定でのお知らせ配信やメニュー表示の切り替え基盤。

### 📱 ユーザーインターフェース
*   **レスポンシブデザイン**: PCおよびモバイル端末への最適化。
*   **管理者/一般ユーザーの切り替え**: 権限に応じたサイドバーメニューとアクセス制限の自動適用。
